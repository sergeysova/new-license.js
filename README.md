# new-license

Generate LICENSE file for your project

[![asciicast](https://asciinema.org/a/164634.png)](https://asciinema.org/a/164634)

## Install

```
$ npm install --global new-license
```

## Usage

```
$ new-license
```

or

```
$ npx new-license
```

You can also use this to update an existing LICENSE file.
